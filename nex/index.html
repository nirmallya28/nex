Job Description

Frontend Developer (React or Angular)

The ideal candidate will be responsible for designing, developing, testing, and debugging responsive web and mobile applications for the company. Using JavaScript, HTML, and CSS, this candidate will be able to translate user and business needs into functional frontend design.


Responsibilities

  • Designing, developing, and testing UI for web applications
  • Build reusable code and libraries for future use
  • Accurately translate user and business needs into functional frontend code

Qualifications

  • 3+ years of experience in frontend development
  • Bachelor’s degree in Computing Science or professional qualifications in frontend development technologies

Skills, Experience and Attitude

  • Identify and troubleshoot UX problems (e.g. responsiveness) with a keen eye for pixel-level detail in UI
  • Understanding of data structures, data modelling and software architecture
  • Deep knowledge of UI/UX design principles and industry standards;
  • Provide thought leadership on Web/Mobile UI experiences with ReactJS, Vue.js, AngularJS and Node JS. Familiarity with other JavaScript frameworks would be a plus.
  • Strong HTML & CSS skills
  • Ability to work in a fast pace delivery oriented environment

Job Description

Full Stack Developer (Java)

We are looking for a self-motivated and creative fulls-stack developer to join our dynamic and experienced team. This position is a hands-on full-stack developer role where you will be empowered to identify, propose, and implement features into our ad tech platform as well as work with our international clients.


Responsibilities and Duties

  • Analyze, design, implement and maintain our cloud based platform.
  • Coding, testing, and support for React, Java based web applications in the cloud environment.

  • 2-3 years of industrial experience in Java development.
  • Hands-on experience in Spring-boot.
  • Experience with HTML, CSS, JavaScript.
  • Experience in ReactJS is an added advantage.
  • Familiarity with Git
  • Cloud infrastructure exposure in Google Cloud and/or AWS
  • Linux operating system experience
  • Database exposure to RDBMS (MySQL)
  • Understanding of Microservices Architecture and Containerization
  • Understanding of Agile and SCRUM practices.
  • Passionate about learning new web technologies
  • Ability to follow coding best practices

  • Work with a very experienced team who are building an innovative ad tech platform
  • Explore and work with new tools, technologies and architectures (React, Spring-boot, Microservices, Google Cloud, Docker, Kubernetes, Serverless, Linux, etc)
  • Exposure to software engineering practises such as coding standards, code reviewing, unit testing, CI/CD, etc
